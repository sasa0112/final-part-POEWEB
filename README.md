1. Introduction

This project is a fully designed and developed website for a fictional bakery called The Bread Basket.
The site was created over three phases, where each part focused on improving different skills such as HTML, CSS, JavaScript, and responsive design.

The final website includes:

A homepage

Product page

About page

Gallery page

Contact page

Enquiry page

The goal of the project was to create a clean, user-friendly, and fully responsive site that demonstrates proper web development practices.
